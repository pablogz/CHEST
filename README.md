# CHEST: Cultural Heritage Educational Semantic Tool

Aplicación distribuida para apoyar a los profesores en el diseño y en la puesta en marcha de situaciones de aprendizaje ubicuas.

Puedes citar nuestro trabajo a través de este artículo

> Pablo García-Zarza, Guillermo Vega-Gorgojo, Miguel L. Bote-Lorenzo, Vanesa Gallego- Lema, Juan I. Asensio-Pérez y Eduardo Gómez-Sánchez. «CHEST: an Application to Support Teachers in the use of Linked Open Data for Ubiquitous Learning». En: IEEE Transactions on Learning Technologies 18 (2025), págs. 1021-1035. doi: [10.1109/tlt.2025.3629584](https://doi.org/10.1109/tlt.2025.3629584)

Esta aplicación fue diseñada y desarrollada en el marco de la tesis doctoral de [Pablo García Zarza](https://scholar.google.es/citations?user=UlVVjUYAAAAJ) dentro del grupo de investigación [GSIC-EMIC](https://gsic.uva.es).
